# EstimationImmo - Toulouse Property Price Prediction

## Description
Modèle de Machine Learning pour estimer le prix de vente des biens immobiliers à Toulouse en utilisant des features créées manuellement.

## Fonctionnalités
- Prédiction du prix de vente basée sur les caractéristiques du bien
- Features engineerisées :
    - Distance au métro le plus proche
    - Surface habitable
    - Nombre de pièces
    - Type de bien

## Prérequis
- Python 3.8+
- pip
